# 市場オッズ超過指数 検証作業

作成日: 2026-07-23

## 目的

地方競馬（local）とJRAの既存SQLiteデータを使い、市場オッズを基準にした市場残差モデル、危険人気馬モデル、調教・血統追加モデルを時系列アウト・オブ・サンプルで検証する。

## 方針

- サーバー上の稼働DBは変更しない。
- `data/` には検証開始時点の読み取り専用スナップショットを置く。
- 閾値は評価期間で選ばず、学習期間内で決定する。
- 最終オッズと購入判断時点オッズを区別する。
- ROIだけでなく、月別安定性、会場別安定性、最大払戻依存、信頼区間を確認する。
- コード・レポートを更新する場合は、作業前バックアップとバージョン更新を行う。

## ディレクトリ

- `data/`: サーバーからコピーしたDB・JSON（Git対象外）
- `scripts/`: 検証コード
- `reports/`: 検証結果
- `backups/`: 編集前バックアップ
